# OpenAI API Example

This repository contains a simple example for using the [OpenAI Node.js SDK](https://github.com/openai/openai-node).

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Set the `OPENAI_API_KEY` environment variable with your API key.
3. Run the example script:
   ```sh
   node OpenAI.js
   ```

The script will print a short bedtime story returned by the OpenAI API. If the environment cannot reach the API, an error message will be shown.
